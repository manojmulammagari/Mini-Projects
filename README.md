# Snake-Water-Gun Game 🐍💧🔫

My first Python project! A text-based game 
similar to Rock, Paper, Scissors.

## Rules
- Snake 🐍 drinks Water 💧 → Snake wins
- Water 💧 ruins Gun 🔫 → Water wins  
- Gun 🔫 shoots Snake 🐍 → Gun wins
- Same choice → Draw

## Features
- Play against computer
- Score tracking for you, computer and draws
- Input validation
- Quit anytime by pressing 'q'

## How to Run
1. Clone this repo:
git clone https://github.com/manojmulammagari/snake-water-gun

2. Run the game:
python main.py

## Concepts Used
- While loops
- If-elif-else conditions
- Dictionaries
- Random module
- User input handling

## How to play:

Run `main.py` in your terminal and type 's', 'w', or 'g' when asked. Type 'q' to quit.
